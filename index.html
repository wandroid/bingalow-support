BinGlow Support

Version 1.0

BinGlow is a binary-style ambient clock display for iPhone.

How to Use BinGlow

BinGlow displays the current time using three rows of lamps:

  • Top row = Hours
  • Middle row = Minutes
  • Bottom row = Seconds

Lamps are binary encoded with the least significant bit (LSB) on the right.

Menu: Long-press anywhere on the screen to open the settings menu.

Subdued Seconds Mode: Seconds are dimmed by default and brighten when motion is detected or at 15-minute intervals.
